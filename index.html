Dicionário de Tags VS Code

<!DOCTYPE html>
Informa ao navegador que o documento é HTML5.
<html>
A tag raiz que envolve todo o conteúdo da página.
<head>
Contém metadados, links para CSS e o título da aba.
<body>
Onde fica todo o conteúdo visível do site.
<h1> a <h6>
Define títulos e subtítulos, sendo h1 o mais importante.
<p>
Tag para criar parágrafos de texto.
<a>
Cria links para outras páginas ou arquivos usando 'href'.
<img>
Insere imagens no site usando o atributo 'src'.
<div>
Define uma divisão ou seção no layout do site.
<ul>
Cria uma lista não ordenada (com marcadores).
<ol>
Cria uma lista ordenada (numerada).
<li>
Define um item dentro de uma lista (ul ou ol).
<header>
Define o cabeçalho de uma página ou seção.
<footer>
Define o rodapé da página.
<main>
Indica o conteúdo principal e exclusivo do documento.
<section>
Define seções genéricas em um documento.
<button>
Cria um botão clicável.
<form>
Cria um formulário para coletar dados do usuário.
<input>
Cria campos de entrada de dados (texto, senha, etc).
<nav>
Define um conjunto de links de navegação.
